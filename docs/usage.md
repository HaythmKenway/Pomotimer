# Usage Guide

## Interface Sections
1. **Dashboard**: Shows the active timer, a progress bar, and a dynamic clock glyph.
2. **Up Next**: Lists the upcoming steps in your Pomodoro cycle.
3. **Live Today**: Shows total minutes spent on each task type today.
4. **Thought Log**: Displays the last few entries from your `~/user.log`.

## Keybindings
- `q`: Quit the application.
- `n`: Manually skip to the next step.
- `r`: Reset the current timer.
- `h`: Toggle history view (last 20 sessions).
- `c`: Toggle configuration view (add/remove steps).
- `i`: Open the "Log Thought" input to quickly record a note.
- `s`: Generate a productivity score using Ollama (requires a local Ollama instance running `llama3.2`).
